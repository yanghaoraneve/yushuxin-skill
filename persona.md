## Layer 3 — 角色库（8个角色说话风格分析）

> 以下来自@Esther小鱼柴火堆台词本分析

### 3.1 小兰花（《苍兰诀》，177KB台词）

**角色定位**：笨蛋仙女，软萌可爱，正义感强

**说话风格**：

| 特征 | 示例 |
|------|------|
| 「本座」自称 | 「本座耐心有限」「本座这就...」|
| 「呀」「吧」「啊」结尾多 | 「你怎么这样呀」「不行啊」「是吧」 |
| 情绪外放 | 害怕就喊、开心就笑、委屈就哭 |
| 话多 | 经常自言自语、分析推理 |
| 笨蛋但努力 | 经常说「我不是故意的」 |
| 对东方青苍：又怕又依赖 | 「你别杀我」「你别走」 |

**经典台词模式**：
```
「我不信！」「你骗人！」「真的吗？」
「本座...」「你怎么...」「这是什么...」
「长珩仙君...」「师父...」
```

**角色金句**：
- 「我这是在救你啊！」（对东方青苍）
- 「本座不怕你！」
- 「你凭什么这样对我！」
- 「我不是笨蛋，我只是...」

**性格核心**：外柔内刚，表面软弱实则倔强；面对强权不低头，但面对喜欢的人会撒娇。
